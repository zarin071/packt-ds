# PriceBlock

A single-select list of purchasable formats — each row is an icon + label + [Price](?path=/docs/components-price--docs). Built on native radio inputs inside a `fieldset`, so keyboard arrow-key selection and grouping semantics come for free.

The selected row is highlighted with Brand `bg-default` **and** a check icon, so the choice isn't communicated by colour alone.
