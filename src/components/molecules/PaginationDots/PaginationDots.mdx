# PaginationDots

Dot navigation for carousels and sliders. Each dot is a real `<button>` with `aria-label="Go to slide N"`, and the active dot carries `aria-current`.

The active dot is also rendered as a wider pill — not just a colour change — so its state isn't communicated by colour alone.
