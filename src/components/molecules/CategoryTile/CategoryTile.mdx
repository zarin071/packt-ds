# CategoryTile

A category entry point. The **whole tile is a single `<a>`**, so it's one keyboard tab stop and one click target — no nested interactive elements to trap focus or fragment the hit area. Composes the [Icon](?path=/docs/components-icon--docs) atom for the glyph.
