# Avatar

A circular user representation built on Radix Avatar — shows an image, initials, or a fallback icon, with an optional status indicator dot.

## Usage

```tsx
import { Avatar } from 'packt-ds';

<Avatar initials="JD" size="medium" />
<Avatar src="/profile.jpg" alt="Jane Doe" size="large" status="online" />
```

Radix's `Avatar.Fallback` only renders once the image has failed to load, or after `fallbackDelayMs` (default `600`) has elapsed without a successful load — so a fast-loading image never flashes initials/icon first.

## Content fallbacks

When no `src` is provided (or the image fails to load) the avatar shows initials; when neither `src` nor `initials` is given it falls back to a generic user icon.
