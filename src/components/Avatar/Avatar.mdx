# Avatar

A circular user representation that displays an image, initials, or a fallback icon — with an optional status indicator dot.

## Usage

```tsx
import { Avatar } from 'packt-ds';

<Avatar initials="JD" size="medium" />
<Avatar src="/profile.jpg" alt="Jane Doe" size="large" status="online" />
```

## Content fallbacks

When no `src` is provided the avatar shows initials; when neither `src` nor `initials` is given it falls back to a generic user icon.
