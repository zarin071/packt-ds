# Button

A token-based button component supporting four semantic variants, three sizes, and both light and dark themes.

## Usage

```tsx
import { Button } from 'packt-ds';

<Button variant="primary" size="md">Get started</Button>
```

## Theming

Wrap the button (or an ancestor) in `data-theme="dark"` when rendering on a dark surface — the component reads from the theme-scoped color tokens automatically, no `theme` prop needed.

```tsx
<div data-theme="dark">
  <Button variant="primary">Button</Button>
</div>
```
