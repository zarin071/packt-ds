# Button

A token-based button component supporting four semantic variants, three sizes, and both light and dark themes.

## Usage

```tsx
import { Button } from 'packt-ds';

<Button variant="primary" size="md" label="Get started" />
```

## Theming

Pass `theme="dark"` when rendering on a dark surface. The component reads from `--packt-semantic-colors-dark-*` tokens automatically.

```tsx
<Button variant="primary" theme="dark" label="Button" />
```
