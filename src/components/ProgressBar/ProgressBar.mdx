# ProgressBar

A determinate progress indicator built on Radix Progress, with three heights and an optional percentage label.

## Usage

```tsx
import { ProgressBar } from 'packt-ds';

<ProgressBar value={65} size="medium" showLabel />
```

The indicator's fill is driven by an inline `transform: translateX(...)` bound to `value` — the same pattern Radix's own docs recommend — while track/fill colors come from Tailwind token utilities.

## Tokens used
