# FormField

[Label](?path=/docs/components-label--docs) + [Input](?path=/docs/components-input--docs) + helper/error text with the ARIA wiring done for you: `htmlFor`/`id` pairing, `aria-describedby` pointing at the helper or error, and `aria-invalid` on error.

Errors render an **icon + text** (`role="alert"`), never colour alone.
