# Select

A token-based single-select dropdown, built on Radix Select for listbox keyboard navigation, focus management, and portal-based positioning.

## Usage

```tsx
import { Select } from 'packt-ds';

<Select
  label="Framework"
  placeholder="Choose a framework…"
  options={[{ value: 'react', label: 'React' }]}
/>
```

## Theming

Wrap the select (or an ancestor) in `data-theme="dark"` when rendering on a dark surface — the component reads from the theme-scoped color tokens automatically, no `theme` prop needed.

```tsx
<div data-theme="dark">
  <Select label="Framework" options={options} />
</div>
```

---
