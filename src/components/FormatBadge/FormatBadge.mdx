# FormatBadge

A [Badge](?path=/docs/components-badge--docs) preset for product formats. Each `format` maps to a fixed icon and label — `ebook`, `paperback`, `video`, `audiobook` — so formats read consistently across the catalogue.

Built entirely on the neutral Badge atom, so it inherits Badge's tokens and theming; see the [Badge](?path=/docs/components-badge--docs) docs for the underlying "Tokens used" table.
