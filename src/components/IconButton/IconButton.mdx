# IconButton

An icon-only button. Because it has no text, `aria-label` is **required and type-enforced** — TypeScript will not compile an `IconButton` without one. The target is 44×44px, meeting the WCAG 2.2 minimum touch-target size.

```tsx
// ✅ compiles
<IconButton aria-label="Close dialog" icon={<CloseIcon />} />

// ❌ TS error: property 'aria-label' is missing
<IconButton icon={<CloseIcon />} />
```
