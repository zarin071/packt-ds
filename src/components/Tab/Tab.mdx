# Tab

A single tab — a styled Radix `Tabs.Trigger` for use inside Radix Tabs. Compose it with the `Tabs`, `TabList`, and `TabPanel` primitives (all exported from the same module). Radix handles roving-tabindex keyboard navigation (arrow keys, Home/End) and the ARIA wiring; the **active tab uses Brand tokens**.

## Usage

```tsx
import { Tabs, Tab, TabList, TabPanel } from 'packt-ds';

<Tabs defaultValue="overview">
  <TabList aria-label="Course sections">
    <Tab value="overview">Overview</Tab>
    <Tab value="curriculum" icon={<FileIcon />}>Curriculum</Tab>
  </TabList>
  <TabPanel value="overview">…</TabPanel>
  <TabPanel value="curriculum">…</TabPanel>
</Tabs>
```
