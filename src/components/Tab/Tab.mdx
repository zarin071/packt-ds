# Tab

A tab list built on Radix Tabs for real roving-tabindex keyboard navigation (arrow keys, Home/End) — renders only the tablist itself, so pair each item's `key` with content you manage elsewhere.

## Usage

```tsx
import { Tab } from 'packt-ds';

<Tab
  items={[
    { key: 'overview', label: 'Overview' },
    { key: 'reviews', label: 'Reviews' },
  ]}
  onChange={(key) => setActiveKey(key)}
/>
```

## Theming

Wrap the tab list (or an ancestor) in `data-theme="dark"` when rendering on a dark surface — the component reads from the theme-scoped color tokens automatically, no `theme` prop needed.

```tsx
<div data-theme="dark">
  <Tab items={items} variant="brand" />
</div>
```

---

## Tokens used
