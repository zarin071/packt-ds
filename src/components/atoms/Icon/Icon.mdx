# Icon

A wrapper that sizes an SVG child (`sm` 16px · `md` 20px · `lg` 24px) and applies `aria-hidden="true"`. The accessible name for an icon-only control must come from the control itself (e.g. `aria-label` on the surrounding button).

## Tokens used

Icon applies **no colour of its own** — the SVG inherits `currentColor` from its parent, so it themes automatically with whatever text colour surrounds it. Sizing uses Tailwind's numeric scale (`size-4` / `size-5` / `size-6`), not a colour token, so there is no light/dark variable to list.
