# Tag

A chip for labels, categories, and filters. Shares its semantic variant set (`brand`, `hub`, `neutral`, `error`, `warning`, `success`, `info`) and token mapping with [Badge](?path=/docs/components-badge--docs); unlike Badge, it can render a removable (×) button via `onRemove`.

## Tokens used
