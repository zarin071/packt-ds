# Tooltip

A hover/focus label built on Radix Tooltip — handles portalling, positioning, delay, and the ARIA wiring for you. Renders on a high-contrast inverted surface so it stands out over any background.

## Tokens used

Every colour is a Tailwind utility wired to tokens in `src/styles/tailwind.css`, which reads the theme-aware aliases in `src/tokens/tokens.theme.css`. Each alias resolves to a different raw variable per `data-theme` — the swatches below show both, resolved live.
