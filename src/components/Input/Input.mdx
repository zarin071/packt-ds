# Input

A bare, styled text field with `default` and `error` states and three sizes. Compose it with the [Label](?path=/docs/components-label--docs) atom and your own helper/error text at the field level — Input owns only its own visual state.

The `error` prop applies error styling and wires `aria-invalid="true"`.

## Tokens used

Every colour is a Tailwind utility wired to tokens in `src/styles/tailwind.css`, which reads the theme-aware aliases in `src/tokens/tokens.theme.css`. Each alias resolves to a different raw variable per `data-theme` — the swatches below show both, resolved live.
