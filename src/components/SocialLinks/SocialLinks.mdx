# SocialLinks

A row of external social links (`github`, `twitter`, `linkedin`, `youtube`). Each link:

- opens in a new tab with `rel="noopener noreferrer"` (no tab-nabbing, no referrer leak),
- includes a visually-hidden **"(opens in new tab)"** in its accessible name so screen-reader users are warned the link leaves the page,
- uses the decorative [Icon](?path=/docs/components-icon--docs) atom for the glyph (aria-hidden).
