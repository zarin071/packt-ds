# RadioButton

A single radio option built on Radix `RadioGroup.Item` + `RadioGroup.Indicator`. Always render it inside a consumer-provided `RadioGroup.Root` (from `@radix-ui/react-radio-group`), which owns the group's `name`/`value`/`onValueChange` — this component renders one option only.

## Tokens used
