# TabGroup

A data-driven wrapper over the Radix [Tab](?path=/docs/components-tab--docs) primitives (`Tabs` / `TabList` / `Tab` / `TabPanel`). Pass an `items` array and it wires up the tablist and panels for you.

Radix provides arrow-key / Home / End navigation and the `tablist` / `tab` / `tabpanel` ARIA roles; the active tab uses **Brand tokens**. If you need finer control, compose the primitives directly instead.

```tsx
import { TabGroup } from 'packt-ds';

<TabGroup
  label="Content type"
  items={[
    { value: 'books', label: 'Books', icon: <BookIcon />, content: <BooksGrid /> },
    { value: 'videos', label: 'Videos', icon: <PlayIcon />, content: <VideosGrid /> },
    { value: 'audiobooks', label: 'Audiobooks', icon: <HeadphonesIcon />, content: <AudioGrid /> },
  ]}
/>
```
