# CountdownTimer

Counts down to a target `Date`, updating every second and rendering days / hours / minutes / seconds.

## Accessibility

The live region is **`aria-live="off"`** by design. A value that changes every second would flood a screen-reader user with announcements — hostile, not helpful — so updates are silent and the current value is available on demand when the user navigates to the `role="timer"`. `onExpire` fires exactly once when the countdown reaches zero.
