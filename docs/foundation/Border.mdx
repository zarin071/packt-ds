# Border

---

## Border Radius

Each swatch is a 48 × 48 square rendered with the actual border-radius value.

<Unstyled>
  <div style={{ fontFamily: 'Outfit, sans-serif', fontSize: 13 }}>
    {[
      { name: 'none',   radius: '0px',   label: '0' },
      { name: 'xs',     radius: '4px',   label: '4px' },
      { name: 's',      radius: '8px',   label: '8px' },
      { name: 'm',      radius: '12px',  label: '12px' },
      { name: 'l',      radius: '16px',  label: '16px' },
      { name: 'pill',   radius: '999px', label: '999px' },
      { name: 'circle', radius: '50%',   label: '50%' },
    ].map(({ name, radius, label }) => (
      <div key={name} style={{ display: 'flex', alignItems: 'center', gap: 20, padding: '10px 12px', borderRadius: 6, marginBottom: 2, background: '#f6f6f6' }}>
        <div style={{
          width: name === 'pill' ? 72 : 48,
          height: 48,
          background: '#f97141',
          borderRadius: radius,
          flexShrink: 0,
        }} />
        <code style={{ flex: 1, fontSize: 12, color: '#4d4d4d' }}>{name}</code>
        <span style={{ width: 44, fontSize: 12, fontWeight: 600, textAlign: 'right', flexShrink: 0 }}>{label}</span>
      </div>
    ))}
  </div>
</Unstyled>

---

## Border Width

Each swatch renders a line at the actual stroke weight.

<Unstyled>
  <div style={{ fontFamily: 'Outfit, sans-serif', fontSize: 13 }}>
    {[
      { name: 'xs', value: 1   },
      { name: 's',  value: 1.5 },
      { name: 'm',  value: 2   },
      { name: 'l',  value: 4   },
      { name: 'xl', value: 8   },
    ].map(({ name, value }) => (
      <div key={name} style={{ display: 'flex', alignItems: 'center', gap: 20, padding: '10px 12px', borderRadius: 6, marginBottom: 2, background: '#f6f6f6' }}>
        <code style={{ width: 24, fontSize: 12, color: '#4d4d4d', flexShrink: 0 }}>{name}</code>
        <div style={{ flex: 1, borderTop: `${value}px solid #f97141` }} />
        <span style={{ width: 44, fontSize: 12, fontWeight: 600, textAlign: 'right', flexShrink: 0 }}>{value}px</span>
      </div>
    ))}
  </div>
</Unstyled>
